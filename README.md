
# ArduinoSerialPackets   [![Badge License]][License]

*Serial transmission library that includes **CRC32** & **ACK**.*

<br>

## Features

-   Reliable Serial Communication

    *Lost & damanged packages are resend.*

-   Maximum Serial Speed
    
    - Software : `19200 bps`
    
    - Hardware : `460800 bps`

-   File Transfer Speed

    `~25 KB / s`

<br>
<br>

## Dependencies

- **[CRC32]**
